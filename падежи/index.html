Падежи

В чувашском языке восемь падежей:

1. Тӗп падеж (основной падеж) — кам? мӗн? (кто? что?)

2. Камӑнлӑх падежӗ (родительный падеж) — камӑн? мӗнӗн? (кого? чего?)

3. Пару падежӗ (дательный падеж) — кама? мӗне? (кому? чему?)

4. Вырӑн падежӗ (местный падеж) — камра? мӗнре? (в ком? в чём?)

5. Туху падежӗ (исходный падеж) — камран? мӗнрен? (из кого? из чего?)

6. Пӗрлелӗх падежӗ (совместный падеж) — кампа? мӗнпе? (с кем? с чем?)

7. Ҫуклӑх падежӗ (лишительный падеж) — камсӑр? мӗнсӗр? (без кого? без чего?)

8. Пирке падежӗ (причинно-целевой падеж) — камшӑн? мӗншӗн? (из-за кого? из-за чего? / для кого? для чего?)

По падежам в чувашском языке изменяются имена существительные и местоимения. Имена прилагательные и имена числительные по падежам не изменяются.
